# Getting help with Headway

## Community support (free)

- **Questions & discussion**: [GitHub Discussions](https://github.com/headway-transit/headway/discussions) (or open an issue if it's a bug).
- **Bugs**: [GitHub Issues](https://github.com/headway-transit/headway/issues) — include record_ids or DQ issue ids where relevant, never raw rider data.
- **Security reports**: see [SECURITY.md](SECURITY.md) — private reporting, please.
- **Docs**: start with the [install guide](install/README.md) and [connecting your data](docs/connecting-your-data.md).

## Commercial support

Professional support, deployment assistance, and agency onboarding are available from **Bekus Solutions**: **support@bekus.co**.

Per the project's [governance charter](GOVERNANCE.md), commercial support is a service around the open platform — it purchases no privilege over the project itself. Every capability supported commercially is the same Apache-2.0 code every agency runs for free, and the roadmap is governed in the open.
